Aprimorando os conhecimentos em HTML 5 e CSS 3


Essa trilha de conhecimento terá como foco revisar e aprimorar os conhecimentos de HTML e CSS, abordando os principais pontos para o desenvolvimento web. Serão trabalhados tópicos como a semântica do HTML, o uso de CSS para criar layouts responsivos (incluindo Flexbox e Grid), e a adoção de boas práticas para garantir acessibilidade, performance e manutenção eficiente dos sites. O objetivo é proporcionar uma base sólida para desenvolver interfaces funcionais e visualmente atrativas, otimizadas para diversos dispositivos.

HTML 5


    HTML (HyperText Markup Language) é a linguagem de marcação usada para estruturar o conteúdo de uma página web, criando elementos como títulos, parágrafos, imagens e links. Ele define a estrutura básica de uma página, permitindo que os navegadores exibam corretamente o conteúdo. É a base de qualquer site, sendo essencial para criar páginas navegáveis e acessíveis.

  • Documentação do HTML

    • [LINK] HTML: Linguagem de Marcação de Hipertexto

    CSS 3


      CSS (Cascading Style Sheets) é a linguagem de estilo usada para definir a aparência visual de uma página HTML. Com CSS, é possível controlar cores, fontes, margens, espaçamentos, posicionamento de elementos e muito mais, criando layouts personalizados e responsivos. CSS permite separar a estrutura (HTML) da apresentação (design), o que facilita a manutenção e evolução do site.
